💡 Exemplos de Uso das APIs

Métricas do Dashboard

GET /api/v1/dashboard-metrics

Obtém todas as métricas do dashboard em tempo real com cache inteligente.

Ver resposta exemplo 
{
"success": true,
"data": {
"documents": {"total": 150, "published": 120},
"audits": {"total": 45, "completed": 40},
"system": {"total_users": 25, "active_users": 20}
},
"cached": true,
"timestamp": "2024-01-01T00:00:00"
}

Predição de Riscos

GET /api/v1/ml/predict/nc-risk?user_id=5

Usa machine learning para predizer risco de ocorrência de NCs.

Ver resposta exemplo 
{
"success": true,
"data": {
"risk_level": "medium",
"confidence": 0.75,
"recommendations": [
  "Aumentar frequência de auditorias internas"
]
}
}

Relatórios Avançados

POST /api/v1/reports/generate/compliance_overview?format=pdf

Gera relatórios profissionais em PDF, Excel, HTML ou JSON.

Ver parâmetros 
{
"start_date": "2024-01-01",
"end_date": "2024-01-31"
}

Portfólio LGPD

GET /api/v1/lgpd/data-portfolio

Obtém portfólio completo de dados pessoais conforme LGPD.

Ver resposta exemplo 
{
"success": true,
"data": {
"personal_information": {...},
"documents_created": 15,
"audit_logs_count": 150,
"consent_records": [...]
}
}
